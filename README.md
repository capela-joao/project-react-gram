# MiniBlog

O **MiniBlog** é um projeto desenvolvido para estudo e prática de **React + TypeScript**, integrando funcionalidades de autenticação, persistência de dados e backend em **Firebase**, **Node.js** e **MongoDB**.  
O objetivo é explorar uma arquitetura moderna, separando frontend e backend, utilizando ferramentas amplamente usadas no mercado.

---

## Tecnologias Utilizadas

### **Frontend**

- React
- TypeScript
- React Router DOM
- CSS Modules
- Firebase Authentication
- Firebase Firestore

### **Outras ferramentas**

- Vite
- Prettier + ESLint
- Git / GitHub
